# Vimkey [![spm version](http://spmjs.io/badge/vimkey)](http://spmjs.io/package/vimkey)

---

Vim-like key mapping from [Vimlide](https://github.com/hotoo/Vimlide).

## Install

```
$ spm install vimkey --save
```

## Usage

```js
var vimkey = require('vimkey');
// use vimkey
```
